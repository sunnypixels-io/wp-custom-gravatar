The WordPress plugin adds a customizable profile picture field to user profiles.

== Description ==
This plugin allows users to upload a custom avatar image. The plugin adds a customizable profile picture field to user profiles. Users can upload a custom avatar image from their profile page. The plugin also provides a shortcode that can be used to display the custom avatar image on the front end.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/sp-custom-gravatar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Users->Your Profile screen to upload a custom avatar image.
4. Enjoy!

== Frequently Asked Questions ==
= How do I upload a custom avatar image? =
You can upload a custom avatar image from your profile page. Go to Users->Your Profile and scroll down to the Custom Avatar section. Click the 'Choose File' button to select an image file from your computer, then click the 'Upload Image' button to upload the image.

= How do I display the custom avatar image on the front end? =
Plugin will replace the default Gravatar image with the custom avatar image.
